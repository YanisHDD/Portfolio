Mon Portfolio

Objectif :

Lors de mon premier stage BTS SIO j'ai eu pour mission de coder mon portfolio et de ne pas utiliser de CMS,
Aidé par l'équipe de developpement Web je l'ai donc mis au point et amélioré au cours de temps

Besoins :

On aura besoin de SublimeText pour coder en HTML, Js, Php, Sass et Css
On aura besoin de la bibliothèque PHPMailer, JQuery
On utilise aussi owl-carousel

Page de présentation :

Je divise la page en plusieurs fichier html et les regroupes dans un fichier php (Pour faciliter le codage)

J'ai d'abord créer un fichier bienvenue.html avec les 3 images du début (Pas sur les images)

Par la suite j'ai créer un fichier pour le menu pour naviguer

Ensuite le fichier moi.html pour me présenter (Pas sur les images)

Le fichier competence.html pour présenter mes competences

Le fichier portfolio.html qui sert à regrouper mes projets

Et enfin le fichier contact.php qui sert à me contacter sur mon mail pro

Le responsive :

J'ai aussi du rendre mon portfolio Responsive suite
C'est à dire qu'il puisse s'adapter au mobile et tablette

Pour réaliser ça j'ai du utliser la règle @media dans le css
Cela consiste à mettre indiquer à partir de quelle dimenssion on modifie le css et comment le modifier par la suite
Dans cet exemple on modifie le css dès que l'on passe sous 1000px de longueur
